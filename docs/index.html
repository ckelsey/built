FunctionToPartial

Converts a function to a curried partial function. Arguments are added by calling the partial(no set arity) until all possible arguments are met, which then calls the original function

FunctionToPartial(function: function (), args: any?): function ()
Parameters
function (function ()) The function to convert
args (any?) The arguments to prepopulate the curried version of the function
Returns
function (): A curried proxy of the passed in function
Example
const curried = FunctionToPartial( (a, b, c) => { console.log(a, b, c) })
curried(`one`) // proxy with argument `a` populated with `one`
curried(`two`) // proxy with argument `b` populated with `two`
curried(`three`) // `one two three`

Get

Searches any type of item down a provided path, returning an emptyVal if provided. Also has the ability to perform a function on the end result

Get(obj: any, path: string, emptyVal: any, modifyFn: function ()): any
Parameters
obj (any) The item to search. Can really be anything, as Get will call methods if provided
path (string) The path chain to follow. Must be dot(.) seperated. Array indice should be dot as well. Can have methods, but arguments can only be strings, numbers, or booleans.
emptyVal (any) If nothing resulted in the search, return this
modifyFn (function ()) A function that will be run to modify the returned result
Returns
any: Either the found item or the emptyVal if provided
Example
const obj = {
     a:{
         b:[
             `C`
         ]
     }
}

Get(obj, `a.b.0.toLowerCase()`) // `c`

SuperFunction

Gives functions super powers. Allows functions to be curried, passing in arguments one at a time, as well as remove or completely replace arguments. Also has Observers to subscribe to for when arguments, curried version are updated as well as when the function is called

SuperFunction(fn: function (), args: Array<any>, name: string?, length: number?): function ()
Parameters
fn (function ()) The function to wrap
args (Array<any>) Arguments to set, for currying
name (string?) Name of the function, curried functions inherit this. Defaults to passed in function's name
length (number?) Arity of function, curried functions inherit this. Defaults to passed in function's length
Properties
args (Array<string>) : Returns or sets the stored arguments
complete (boolean) : Returns true if the stored arguments length is that of the original functions argument length
curried (function) : Returns a new super copy of the function
curry (function (...args)) : Returns a copy of the current curried function, adding in any passed arguments
pushArgument (function (arg)) : Adds a new argument
popArgument (function ()) : Removes the last argument
subscribe (function (property: string, next: function, error: function, complete: function)) : Subscribes to a state property observer, if it exists
Returns
function (): A proxy version of the original function with the above properties/methods
Example
const wonderFunction = SuperFunction( (a, b, c) => `${a} | ${b} | ${c}` )

wonderFunction(`one`) // `one undefined undefined`

const curried = wonderFunction.curried
curried(`one`) // proxy with argument `a` populated with `one`
curried(`two`) // proxy with argument `b` populated with `two`
curried(`three`) // `one | two | three`

const curried2 = wonderFunction.curried
curried2.pushArgument(`Aquaman`) // (`Aquaman`, b, c) => `${a} | ${b} | ${c}`

const curried3 = curried2.curried
curried3.pushArgument(`Skateman`) // (`Aquaman`, `Skateman`, c) => `${a} | ${b} | ${c}`
curried3.length // 3
curried3.args // [`Aquaman`, `Skateman`]
curried3.complete // false
curried3.versions // [curried, curried2]

curried3.popArgument()
curried3.args // [`Aquaman`]

curried3.curry(`Krypto the Superdog`, `Arm Rip Off Boy`) // `Aquaman | Krypto the Superdog | Arm Rip Off Boy`

curried3.subscribe(`args`, console.log)
curried3.pushArgument(`Squirrel Girl`) 
// subscription triggered -> [`Aquaman`, `Squirrel Girl`]

curried3.subscribe(`call`, console.log)
curried3(`Razorback`)
// subscription triggered -> {
//   arguments:[
//       `Aquaman`, 
//       `Squirrel Girl`, 
//       `Razorback`
//   ], 
//   returned: `Aquaman | Squirrel Girl | Razorback`, 
//   self:curried3 
// }
